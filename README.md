# TypeApp
TypeApp is a simple yet highly functional text editor made with python and tkinter.

## Features
- Periodic AutoSave
- Find & Replace 
- Word Count 
- Text Customization 
- Highlighting 
- Integrated Clock 
- Theme Customization
- File Handling


## Dependencies
- Python 3.x
- Tkinter library (usually included with Python installation)


## Installation

To install TypeApp

```bash
  git clone https://github.com/ChiragKt675/TypeApp

  cd TypeApp
  
  python3 TypeApp.py
```
